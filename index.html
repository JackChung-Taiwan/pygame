main.py
import pygame

pygame.init()
screen = pygame.display.set_mode((800, 600))
clock = pygame.time.Clock()
running = True

while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((32, 76, 120))
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
▶ 遊戲視窗已啟動，FPS：60

LESSON 02

理解遊戲主迴圈

遊戲會不斷執行「事件 → 更新 → 繪製 → 控制 FPS」。

clock = pygame.time.Clock()
running = True

while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((245, 249, 255))

實作任務：修改背景 RGB 顏色，觀察遊戲視窗的變化。
LESSON 03

使用鍵盤控制角色

get_pressed() 能在按住按鍵時持續移動角色。

keys = pygame.key.get_pressed()

if keys[pygame.K_LEFT] or keys[pygame.K_a]:
    player.x -= speed
if keys[pygame.K_RIGHT] or keys[pygame.K_d]:
    player.x += speed
if keys[pygame.K_UP] or keys[pygame.K_w]:
    player.y -= speed
if keys[pygame.K_DOWN] or keys[pygame.K_s]:
    player.y += speed
實作任務：加入 Shift 加速功能，讓玩家移動速度變成兩倍。
LESSON 04

繪製圖形與載入圖片

可先用幾何圖形製作原型，再替換成圖片素材。

# 畫出矩形玩家
pygame.draw.rect(screen, (55, 118, 171), player)

# 載入透明 PNG
image = pygame.image.load("player.png").convert_alpha()
image = pygame.transform.scale(image, (64, 64))
screen.blit(image, player)
實作任務：用圓形製作金幣，再使用黃色填滿。
LESSON 05

碰撞判定與分數

利用 Rect 判斷玩家是否碰到食物、敵人或終點。

if player.colliderect(coin):
    score += 1
    coin.x = random.randint(0, WIDTH - coin.width)
    coin.y = random.randint(0, HEIGHT - coin.height)

player.clamp_ip(screen.get_rect())
實作任務：加入生命值，碰到紅色敵人時扣除一點生命。
LESSON 06

音效、音樂與遊戲完成

加入聲音回饋，並設計開始、遊玩、結束與重新開始狀態。

coin_sound = pygame.mixer.Sound("coin.wav")
coin_sound.set_volume(0.5)
coin_sound.play()

pygame.mixer.music.load("music.ogg")
pygame.mixer.music.play(-1)
實作任務：按 R 重新開始遊戲，按空白鍵暫停或繼續。

FAQ

常見問題

輸入 pip install pygame 後仍然無法 import，怎麼辦？

確認安裝套件與執行程式使用相同的 Python。建議改用 python -m pip install pygame，再用 python 檔名.py 執行。

為什麼按方向鍵沒有反應？

先點一下遊戲視窗，使它取得鍵盤焦點；並確認遊戲迴圈中有持續執行 pygame.event.get()。

KEYDOWN 和 get_pressed() 有什麼不同？

KEYDOWN 適合單次操作，例如跳躍或暫停；get_pressed() 適合按住後持續移動。

如何避免玩家跑出遊戲畫面？

建立玩家 Rect 後執行 player.clamp_ip(screen.get_rect())，即可把玩家限制在視窗範圍內。
